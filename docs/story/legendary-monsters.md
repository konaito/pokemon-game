# 伝説のモンスター設計書

## コンセプト

Monster Chronicleの伝説のモンスターは、テーマ「忘却と記憶」の神話的象徴。
2体の対の存在が世界のバランスを保っていたが、人間の介入により均衡が崩れた。

---

## オモイデ（想出）— 記憶の守護者

### 基本情報

| 項目   | 内容                           |
| ------ | ------------------------------ |
| 名前   | オモイデ                       |
| 由来   | 想い出（おもいで）— 大切な記憶 |
| タイプ | エスパー / フェアリー          |
| 分類   | きおくモンスター               |
| 高さ   | 2.1m                           |
| 重さ   | 52.0kg                         |

### 外見コンセプト

- **モチーフ**: 鳳凰 × 水晶 × 虹
- 体の一部が透明な結晶でできており、中に虹色の光が宿っている
- 翼は薄く半透明で、光を受けると虹のスペクトルに分かれる
- 穏やかで慈愛に満ちた表情。目は深い紫色
- 尾羽は長く、先端に小さな光の珠が連なる（記憶の「連鎖」を象徴）

### 種族値

| HP  | 攻撃 | 防御 | 特攻 | 特防 | 素早 | 合計 |
| --- | ---- | ---- | ---- | ---- | ---- | ---- |
| 100 | 65   | 90   | 130  | 120  | 95   | 600  |

- **特攻・特防型**: 記憶を操る精神の力が最も強い
- HPも高く、守護者としての耐久性

### 能力

- 生き物の記憶を守り、蘇らせる力を持つ
- 近くにいると忘れていた記憶が断片的に蘇ることがある
- 結晶の体に世界中の記憶の破片を宿している

### 物語上の役割

- セイレイ山の祠に眠っていたが、主人公の行動に呼応して目覚める
- メインストーリーのクライマックスで主人公と共闘し、ワスレヌの暴走を止める
- メインストーリーでは捕獲せず共闘のみ。ポストゲームで捕獲イベント

---

## ワスレヌ（忘れぬ → 反転して忘却）— 忘却の存在

### 基本情報

| 項目   | 内容                                                                 |
| ------ | -------------------------------------------------------------------- |
| 名前   | ワスレヌ                                                             |
| 由来   | 忘れぬ（わすれぬ）→ 本来は「忘れない」という意味。皮肉にも忘却を司る |
| タイプ | エスパー / あく                                                      |
| 分類   | ぼうきゃくモンスター                                                 |
| 高さ   | 2.3m                                                                 |
| 重さ   | 68.0kg                                                               |

### 外見コンセプト

- **モチーフ**: 黒い霧 × 蛾（月に惹かれる虫）× 朧月
- 体から黒い霧を纏う。輪郭が曖昧で、見ていると記憶が薄れるような不安感
- 翼は蛾のように大きく丸みがあるが、模様は目のような渦巻き
- 目は虚ろな金色。悲しげだが、攻撃的ではない
- 額に三日月のような紋章。これが暴走の印

### 種族値

| HP  | 攻撃 | 防御 | 特攻 | 特防 | 素早 | 合計 |
| --- | ---- | ---- | ---- | ---- | ---- | ---- |
| 100 | 130  | 80   | 120  | 80   | 90   | 600  |

- **攻撃・特攻型**: 暴走した力は攻撃的に発現している
- 防御・特防が低いのは「内部のバランスが崩れている」ことの象徴

### 能力

- 記憶を消し去る力。本来は「辛い記憶を和らげる」慈悲の力
- 50年前に力を反転させられ、世界規模の記憶消去（大忘却）を引き起こした
- 現在も暴走状態が続いており、近くにいると記憶が曖昧になる

### 物語上の役割

- 50年前、研究者ムゲンに利用されて大忘却を引き起こした被害者でもある
- オブリヴィオン団がその力を再度利用しようとしている
- クライマックスでは敵として戦うが、倒すことで暴走が止まる
- ポストゲームで暴走が解け、本来の穏やかな姿に戻る。この状態で捕獲可能

## 二体の関係 — 対としてのバランス

```
        ┌──────────────────────────────┐
        │     世界の記憶のバランス     │
        │                              │
        │  オモイデ ←───→ ワスレヌ    │
        │  （守る）        （解放する） │
        │  （記録する）    （忘れさせる）│
        │  （光）          （影）       │
        │                              │
        │  フェアリー ←→ あく          │
        │  （物語）        （忘却）     │
        └──────────────────────────────┘
```

### 本来の役割

- **オモイデ**: 大切な記憶を守り、伝承として次世代に残す
- **ワスレヌ**: 辛い記憶、トラウマ、悲しみを和らげ、人が前に進めるようにする
- 二体で「覚えることと忘れること」のバランスを保ち、生命の精神的健全を守っていた

### 大忘却の経緯

1. 50年前、研究者ムゲンがワスレヌの力に着目
2. 「記憶が争いの原因」→ 全ての記憶を消せば平和になると考えた
3. ワスレヌの力を増幅・反転させ、選択的な忘却を無差別な忘却に変えた
4. 大忘却が発生。人間とモンスターの絆の記憶が世界規模で消えた
5. ワスレヌ自身も暴走により自我を失い、オモイデは力を失って眠りについた

### テーマとの関係

> 「覚えること」と「忘れること」は、どちらも人間に必要な力。
> 全てを覚えていたら苦しみに押し潰される。全てを忘れたら自分が誰かも分からなくなる。
> 大切なのは、**何を覚え、何を手放すかを自分で選べること**。

## 神話テキスト（ゲーム内で遺跡や古文書に記される）

### オモイデの神話

> 「はるか昔、虹色の鳥が空を舞った。
> その翼に触れた者は、忘れかけた大切な記憶を取り戻したという。
> 人はその鳥を『オモイデ』と呼び、記憶の守り手として崇めた。」

### ワスレヌの神話

> 「虹色の鳥と共に、月色の蛾が舞った。
> その霧に包まれた者は、辛い過去を忘れ、穏やかに眠れたという。
> 人はその蛾を『ワスレヌ』と呼んだ。忘れぬ想いを、静かに解放する者。」

### 対の神話

> 「記憶の鳥と忘却の蛾は、二つで一つ。
> 覚えることと忘れること。光と影。
> どちらか一方が欠ければ、世界は狂う。」
