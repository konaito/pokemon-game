# エンディングとポストゲームの設計書

## エンディングシーケンス

### 前提条件

- 全8ジムバッジ取得済み
- 四天王4人 + チャンピオン・アカツキを撃破済み
- メインストーリー（オブリヴィオン団壊滅、ワスレヌの暴走停止）完了済み

### シーケンス

```
1. チャンピオン戦勝利
   └─ アカツキとの最終バトルに勝利

2. 殿堂入りの儀式
   └─ パーティ6匹の名前とレベルが記録される
   └─ 「{プレイヤー名}は殿堂入りした！」

3. アカツキとの会話（クライマックス）
   └─ 「私はあの日、全てを諦めた。だがお前は諦めなかった。
       ...ありがとう。お前のおかげで、私もまた前に進める」

4. スタッフロール
   └─ BGM: 特別エンディング曲
   └─ 旅の回顧映像（各町の風景、ジムリーダーとの出会い）
   └─ 各キャラクターの「その後」の一枚絵

5. エピローグ
   └─ コダマ町の自宅に戻る
   └─ 母親「おかえり。立派になったね」
   └─ コダチ博士「世界は変わり始めている。記憶は戻らなかった。
       だが...新しい物語が始まっている」

6. ソウマとの最後の会話
   └─ 「...俺もモンスターの研究をしてみようと思う。
       お前が見せてくれたもの、ちゃんと覚えてるから」

7. エンディングメッセージ
   └─ 「あなたの旅は、ここで一つの区切りを迎えます。
       しかし、レムニアの物語はまだ続いています。
       新しい冒険が、あなたを待っています。」
```

### 各キャラクターの「その後」

| キャラクター         | エンディング後の状況                                       |
| -------------------- | ---------------------------------------------------------- |
| **主人公**           | チャンピオンの座に就くが、引き続き旅を続けることを選ぶ     |
| **ソウマ**           | モンスター研究者の道を志し、コダチ博士の助手になる         |
| **コダチ博士**       | 大忘却の記録をまとめた論文を発表。世界的に注目される       |
| **アカツキ**         | チャンピオンの座を譲り、再び旅に出る（ポストゲームで再会） |
| **カゲロウ**         | 逮捕後、自らの過ちを認め始める（ポストゲームで面会可能）   |
| **ジムリーダーたち** | 各地で「モンスターとの交流の場」を開設し始める             |

## ポストゲーム

### 概要

エンディング後、主人公はコダマ町から再び自由に冒険できる。
メインストーリーでは行けなかった場所や、残された謎を解き明かすコンテンツ。

### コンテンツ一覧

#### 1. 伝説のモンスター捕獲イベント

**オモイデ捕獲**

- 場所: セイレイ山の最深部（メインストーリーで共闘した場所）
- 条件: 殿堂入り後、再びセイレイ山に向かう
- 演出: オモイデが主人公を認め、バトルで実力を試す
- レベル: 70

**ワスレヌ捕獲**

- 場所: 忘却の遺跡の最深部
- 条件: オモイデ捕獲後、忘却の遺跡の封印が解ける
- 演出: 暴走が解けた本来の穏やかなワスレヌと出会う
- レベル: 70

#### 2. 遺跡完全調査

「大忘却の全容」を解明するサブクエスト。

| 遺跡               | 内容                     | 報酬             |
| ------------------ | ------------------------ | ---------------- |
| 忘却の遺跡・深層   | ムゲンの研究日記を発見   | 技マシン         |
| アサヒ海底洞窟・奥 | 古代の壁画「共存の時代」 | レアアイテム     |
| キリフリの秘密の間 | 大忘却の目撃者の記録     | 技マシン         |
| セイレイ山・頂上   | オモイデとワスレヌの祠   | 伝説専用アイテム |

#### 3. アカツキ再戦

- 場所: ワスレナ島のチャンピオンロード奥
- 条件: 殿堂入り後
- パーティ: 全モンスターLv75以上の強化版
- アカツキ「あの時の借りを返しに来たぞ。今度は全力でいかせてもらう」

#### 4. オブリヴィオン団残党イベント

- カゲロウ面会: 収監施設で面会可能。大忘却の詳細を語る
- 幹部トコヤミ: 改心して研究者として活動。情報提供者になる
- 残党掃討: 各地に散った団員を捜索する小イベント

#### 5. 図鑑完成への道

- 全モンスター（序盤+中盤+終盤+伝説）の捕獲を目指す
- 図鑑完成報酬: コダチ博士から特別な称号

#### 6. ソウマ最終バトル

- 場所: コダマ町（旅の始まりの場所）
- 条件: 全ジム再戦クリア後
- ソウマのフルパーティ（6匹Lv65以上）との友情バトル
- 「全力で来い。お前にそう言えるようになった自分が、少し誇らしい」

## 設計方針

### ポストゲームのテーマ

メインストーリーのテーマ「忘却と記憶」の延長として:

- **知る**: 大忘却の全容を解明する（遺跡調査）
- **受け入れる**: 敵だったカゲロウの視点を理解する
- **前に進む**: アカツキ・ソウマとの再戦は「新しい関係」の象徴

### ボリューム感

- メインストーリー: 約20-30時間
- ポストゲーム: 約10-15時間
- 図鑑完成: +10時間程度

### 難易度

ポストゲームのバトルはメインストーリーより明確に難しい:

- 伝説モンスター: Lv70（メインストーリー終了時の平均は50-55程度）
- アカツキ再戦: Lv75
- 育成と戦略の工夫が必要になる
