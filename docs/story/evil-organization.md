# 悪の組織設計書 — オブリヴィオン団

## 概要

| 項目     | 内容                                                         |
| -------- | ------------------------------------------------------------ |
| 名称     | オブリヴィオン団 (Team Oblivion)                             |
| 語源     | Oblivion = 忘却、消滅                                        |
| シンボル | 割れた鏡（反射=記憶が砕ける様）                              |
| 理念     | 「忘却は進化。絆は呪い」                                     |
| 目的     | 大忘却を完全なものにし、人間をモンスターから完全に独立させる |
| 手段     | ワスレヌの力を再発動させ、残存する記憶も消去する             |

---

## ボス: カゲロウ

### プロフィール

- **名前由来**: 陽炎（実体のない幻影）/ 蜉蝣（短命の虫）
- **年齢**: 50歳
- **外見**: 長身痩躯。銀髪。割れた鏡のペンダント
- **口調**: 静かで知的。怒鳴ることはない

### 背景

50年前、大忘却を起こした研究者 **ムゲン** の弟子。ムゲンは「人間とモンスターの共存が戦争の原因になっている」と信じ、記憶を消すことで平和を実現しようとした。結果として世界は「平和」になったが、それは忘却による無関心に過ぎなかった。

カゲロウはムゲンの理念を純粋に信じている。彼にとって忘却は「善」であり、記憶を取り戻そうとする者は「世界を再び混乱に陥れる愚か者」。

### 動機の深層

カゲロウ自身も大忘却の被害者。彼はかつてモンスターと深い絆を持っていたが、その記憶が消された後の「喪失感」に苦しんだ。

> 「覚えていないのに、何かを失ったことだけは分かる。この空虚が...記憶を持つ者なら、もっと苦しいはずだ。ならば全てを消すことが慈悲だ」

### パーティ

エスパー/あくタイプを中心としたパーティ。「記憶操作」を暗示。

## 幹部

### ウツロ（虚）— 実行部隊長

- **テーマ**: 空虚
- **背景**: 大忘却で最も多くの記憶を失った人物の一人。感情が希薄で、命令に従うことだけが存在意義
- **物語上の機能**: 「記憶を失うとはどういうことか」を具体的に見せる装置
- **バトルスタイル**: あくタイプ中心。冷酷で無駄のない戦い方
- **遭遇**: 3回（ナツメ町、キリフリ村、セイレイ山）

### マボロシ（幻）— 情報操作担当

- **テーマ**: 幻影
- **背景**: 元心理学者。偽の記憶を植え付ける技術を持つ。人々に「モンスターは昔から危険だった」と信じ込ませる
- **物語上の機能**: 「記憶が操作される恐怖」を提示。信頼できるものは何か？
- **バトルスタイル**: エスパー/ゴーストタイプ。幻惑する戦い方
- **遭遇**: 3回（カガリ市、忘却の遺跡、セイレイ山）

### トコヤミ（常闇）— 研究部門長

- **テーマ**: 永遠の闇
- **背景**: ワスレヌの力を研究する科学者。純粋な知的好奇心で忘却の力を解明しようとしている。カゲロウの理念にはあまり興味がなく、研究のために団にいる
- **物語上の機能**: 「善悪の境界が曖昧な存在」。倒した後も情報を提供してくれる
- **バトルスタイル**: エスパー中心。計算尽くの戦略的な戦い方
- **遭遇**: 2回（忘却の遺跡、セイレイ山）

- **外見**: 黒と銀の制服。割れた鏡のエンブレム

## テーマの「裏返し」としてのオブリヴィオン団

主人公のテーマ: 「記憶がなくても新しい絆を作れる」
オブリヴィオンのテーマ: 「絆そのものが苦しみの源」

両者は「記憶の喪失」を経験しているが、そこから導く結論が正反対:

```
主人公:  失った → でも新しく作れる → 前に進む
カゲロウ: 失った → だから全部消す → 痛みをなくす
```

プレイヤーへの問い: **「痛みを避けるために全てを手放すのか、痛みを受け入れて新しいものを掴むのか」**
