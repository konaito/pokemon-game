# リージョン設計書 — レムニア地方

## コンセプト

**レムニア地方** (Remnea Region)

- 語源: Remnant（残存物）+ Memoria（記憶）
- テーマ: 「忘却された絆の痕跡が眠る土地」
- 地理モチーフ: 日本の中国・四国地方をベースにした温暖な半島地域
- 特徴: かつてモンスターと人間が共存していた「古代遺跡」が各地に点在する

### 気候のグラデーション

- **南部**: 温暖な海洋性気候。明るく開放的な序盤
- **中央**: 温帯。穏やかだが、森や洞窟に謎が潜む
- **北部**: 寒冷。厳しい自然。物語のクライマックス
- **離島**: 風が強く、隔絶された空気。最終試練

## 文化・社会

### 大忘却以前

- 人間とモンスターが共に暮らす「共存時代」
- 各町にモンスターと人間が一緒に働く姿があった
- 遺跡に残る壁画がその様子を伝えている

### 大忘却以後（現在）

- モンスターは「野生動物」として恐れられている
- 一部の研究者と特殊な人々だけがモンスターと関わっている
- 古い世代は「あの頃は良かった」と語るが、若い世代は信じていない
- ジムリーダーたちは「古い絆」を密かに守り続けてきた「守り手」

### 技術レベル

- 現代的だが、モンスターボールは「失われた技術」を博士が再現した
- ポケモンセンターは研究施設を兼ねる（一般の病院ではモンスターを診ない）
- 通信システムは発達しているが、モンスター関連の知識は失われている
