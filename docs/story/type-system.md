# タイプシステム正式決定

## 方針: 本家18タイプを採用

Monster Chronicleでは、ポケモン本家の18タイプシステムをそのまま採用する。

### 理由

1. **実証済みのバランス**: 25年以上のプレイテストに裏打ちされた相性バランス
2. **認知負荷の最適化**: 18タイプは「覚えられる上限」と「深みの下限」の絶妙なバランス
3. **段階的学習**: 草/火/水の3すくみから始まり、徐々に複雑さが増す設計
4. **既に実装済み**: コードベースのタイプ相性テーブルが18タイプ対応

## タイプとテーマの関係

Monster Chronicleでは、タイプは単なるバトル属性ではなく、「記憶と忘却」のテーマと結びつく。

### 伝説のモンスターのタイプ

- **オモイデ**: エスパー/フェアリー — 記憶(精神) × 物語(幻想)
- **ワスレヌ**: エスパー/あく — 記憶(精神) × 忘却(闇)

エスパーは両者に共通し、「記憶を操る力」の源。フェアリーとあくが対になる。

### ジムの順番とタイプ選択理由

1. **むし**: 序盤に適した低難易度。「小さな命」から始まる
2. **みず**: 海の町。世界の広がりを感じさせる
3. **くさ**: 自然との調和。壁画の発見と連動
4. **ほのお**: 都市。人間の文明側。悪の組織登場と連動
5. **じめん**: 遺跡の町。地下に眠る真相
6. **ゴースト**: 霧の村。死者の記憶。物語が深くなる
7. **こおり**: 雪山。凍りついた（保存された）記憶
8. **ドラゴン**: 古都。最も古い絆の象徴

## 拡張方針

現時点では新タイプの追加は予定しない。ただし将来的に以下の拡張を検討:

- **記憶タイプ（Memory）**: ポストゲーム/DLCで追加の可能性。伝説のモンスター専用
- ただし18タイプのバランスを壊すリスクがあるため、慎重に検討
